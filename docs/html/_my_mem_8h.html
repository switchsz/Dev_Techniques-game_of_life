Detailed Description

Memory management functions (user interface). Two versions of memory allocation/deallocation are available. If compiled with MEMTRACE defined, CUnit keeps track of all system allocations & deallocations. The memory record can then be reported using CU_CREATE_MEMORY_REPORT. Otherwise, standard system memory allocation is used without tracing.
