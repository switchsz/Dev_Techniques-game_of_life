Detailed Description

Management functions for tests, suites, and the test registry. Unit testing in CUnit follows the common structure of unit tests aggregated in suites, which are themselves aggregated in a test registry. This module provides functions and typedef's to support the creation, registration, and manipulation of test cases, suites, and the registry.
