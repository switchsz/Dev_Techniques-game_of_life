Create a C\+A\+Metal\+Layer-\/backed N\+S\+View/\+U\+I\+View and attach it to the specified window. 

On mac\+OS, this does {\itshape not} associate a M\+T\+L\+Device with the C\+A\+Metal\+Layer on its own. It is up to user code to do that.

The returned handle can be casted directly to a N\+S\+View or U\+I\+View, and the C\+A\+Metal\+Layer can be accessed from the view\textquotesingle{}s \textquotesingle{}layer\textquotesingle{} property.
