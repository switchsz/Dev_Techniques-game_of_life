Runs C\+Unit tests using the automated interface. This function sets appropriate callback functions, initializes the test output files, and calls the appropriate functions to list the tests and run them. If an output file name root has not been specified using \doxyref{C\+U\+\_\+set\+\_\+output\+\_\+filename()}{p.}{group___automated_ga60583b154146b787f03267fc320f86e5}, a generic root will be applied. It is an error to call this function before the C\+Unit test registry has been initialized (check by assertion). \mbox{\label{group___automated_ga4bf1c3a0b54cb2a12d634e7e022edf30}}
