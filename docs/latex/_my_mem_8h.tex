\doxysubsection{Detailed Description}
Memory management functions (user interface). Two versions of memory allocation/deallocation are available. If compiled with M\+E\+M\+T\+R\+A\+CE defined, C\+Unit keeps track of all system allocations \& deallocations. The memory record can then be reported using C\+U\+\_\+\+C\+R\+E\+A\+T\+E\+\_\+\+M\+E\+M\+O\+R\+Y\+\_\+\+R\+E\+P\+O\+RT. Otherwise, standard system memory allocation is used without tracing.
