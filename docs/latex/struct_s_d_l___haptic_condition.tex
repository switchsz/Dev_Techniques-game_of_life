\doxysubsection{Detailed Description}
A structure containing a template for a Condition effect. 

The struct handles the following effects\+:
\begin{DoxyItemize}
\item \doxyref{S\+D\+L\+\_\+\+H\+A\+P\+T\+I\+C\+\_\+\+S\+P\+R\+I\+NG}{p.}{_s_d_l__haptic_8h_a3b52d4700380085e2b5d87bb20320fb1}\+: Effect based on axes position.
\item \doxyref{S\+D\+L\+\_\+\+H\+A\+P\+T\+I\+C\+\_\+\+D\+A\+M\+P\+ER}{p.}{_s_d_l__haptic_8h_a69522f261973e3ea4273486141320a87}\+: Effect based on axes velocity.
\item \doxyref{S\+D\+L\+\_\+\+H\+A\+P\+T\+I\+C\+\_\+\+I\+N\+E\+R\+T\+IA}{p.}{_s_d_l__haptic_8h_a14b807471589120395aec28293cf6642}\+: Effect based on axes acceleration.
\item \doxyref{S\+D\+L\+\_\+\+H\+A\+P\+T\+I\+C\+\_\+\+F\+R\+I\+C\+T\+I\+ON}{p.}{_s_d_l__haptic_8h_a98a5011311168d63921aaac21403a4d8}\+: Effect based on axes movement.
\end{DoxyItemize}

Direction is handled by condition internals instead of a direction member. The condition effect specific members have three parameters. The first refers to the X axis, the second refers to the Y axis and the third refers to the Z axis. The right terms refer to the positive side of the axis and the left terms refer to the negative side of the axis. Please refer to the \doxyref{S\+D\+L\+\_\+\+Haptic\+Direction}{p.}{struct_s_d_l___haptic_direction} diagram for which side is positive and which is negative.
