\doxysubsection{Detailed Description}
Include file for S\+DL game controller event handling

In order to use these functions, \doxyref{S\+D\+L\+\_\+\+Init()}{p.}{_s_d_l_8h_a8fc8d35348d7c74bad8392d776c937b8} must have been called with the \doxyref{S\+D\+L\+\_\+\+I\+N\+I\+T\+\_\+\+G\+A\+M\+E\+C\+O\+N\+T\+R\+O\+L\+L\+ER}{p.}{_s_d_l_8h_ac224a9aa4208de0e9b03948ff9b5fe0a} flag.
