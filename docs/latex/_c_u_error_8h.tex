\doxysubsection{Detailed Description}
Error handling functions (user interface). C\+Unit uses a simple (and conventional) error handling strategy. Functions that can generate errors set (and usually return) an error code to indicate the run status. The error code can be inspected using the \doxyref{C\+U\+\_\+get\+\_\+error()}{p.}{group___framework_ga0bf3f9733e5fa0c0ed6ca0a37a2d995e} function. A descriptive error message can be retrieved using \doxyref{C\+U\+\_\+get\+\_\+error\+\_\+msg()}{p.}{group___framework_gaee27bf496da77437288bc9fe2a0b20ed}.
