\doxysubsection{Detailed Description}
The generic template for any haptic effect. 

All values max at 32767 (0x7\+F\+FF). Signed values also can be negative. Time values unless specified otherwise are in milliseconds.

You can also pass \doxyref{S\+D\+L\+\_\+\+H\+A\+P\+T\+I\+C\+\_\+\+I\+N\+F\+I\+N\+I\+TY}{p.}{_s_d_l__haptic_8h_a648896e9e14cb27fa8a4b6632d3f3882} to length instead of a 0-\/32767 value. Neither delay, interval, attack\+\_\+length nor fade\+\_\+length support \doxyref{S\+D\+L\+\_\+\+H\+A\+P\+T\+I\+C\+\_\+\+I\+N\+F\+I\+N\+I\+TY}{p.}{_s_d_l__haptic_8h_a648896e9e14cb27fa8a4b6632d3f3882}. Fade will also not be used since effect never ends.

Additionally, the \doxyref{S\+D\+L\+\_\+\+H\+A\+P\+T\+I\+C\+\_\+\+R\+A\+MP}{p.}{_s_d_l__haptic_8h_af10eb937a64a8f602e9c46682ac0d868} effect does not support a duration of \doxyref{S\+D\+L\+\_\+\+H\+A\+P\+T\+I\+C\+\_\+\+I\+N\+F\+I\+N\+I\+TY}{p.}{_s_d_l__haptic_8h_a648896e9e14cb27fa8a4b6632d3f3882}.

Button triggers may not be supported on all devices, it is advised to not use them if possible. Buttons start at index 1 instead of index 0 like the joystick.

If both attack\+\_\+length and fade\+\_\+level are 0, the envelope is not used, otherwise both values are used.
